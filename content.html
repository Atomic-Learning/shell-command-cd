The cd command is used to change the current directory of the shell. It is available in shells in all Unix-like systems and some others, including:

  • bash
  • zsh
  • PowerShell (technically an alias for Set-Location)

Usage

The basic syntax of the cd command is cd followed by a path to the desired directory. This is most commonly a relative path, but can be an absolute path. cd .. allows you to move up one level in the directory tree.

Changing to the Home Directory

In most shells, you can use the cd command without any arguments or with a tilde cd ~ as a shortcut to change to the home directory.
